# 📱 TimePilot-App

> Android client of **TimePilot**, a lightweight and elegant time management & learning plan application.

---

## 🇨🇳 简体中文版本

### 🚀 项目简介

`TimePilot-App` 是 **TimePilot 时间管理系统** 的移动端应用，  
帮助用户高效记录待办事项、规划学习计划、追踪习惯完成度。

本应用注重极简设计与流畅体验，采用 **Kotlin + Jetpack Compose + MVVM 架构**，  
通过调用 `TimePilot-Backend` 提供的 API 实现数据交互与账户认证。

#### 🎯 核心目标
- 🧭 提供极简高效的日程管理体验  
- 🔔 支持计划提醒、习惯追踪与时间分析  
- ☁️ 支持云端同步与多端数据共享  
- 🛡️ 确保用户数据安全与隐私保护
